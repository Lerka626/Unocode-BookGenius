# Unocode BookGenius

Сравнение эффективности LightGBM и Llama 3 в задаче классификации жанров книг по аннотациям

## Описание проекта

В современном мире автоматическая классификация текстов — востребованная задача. В этом проекте мы сравниваем два подхода к определению жанра книги по её краткому описанию:

- Классический ML-пайплайн – TF‑IDF + LightGBM (обучен на 15 000 аннотаций).
- Современная LLM – Llama 3 в режиме zero-shot (без дообучения).

Итоговый сервис представляет собой FastAPI-приложение, которое:
- принимает текст аннотации;
- предсказывает жанр (один из 10 самых частых);
- возвращает объяснение решения (ключевые слова для классической модели, текстовое пояснение от LLM);
- позволяет сравнивать подходы в реальном времени через единый API.

## Почему это важно?

| Критерий | LightGBM + TF‑IDF | Llama 3 (zero-shot) |
|----------|-------------------|----------------------|
| Точность | 77.94%| ~60–70% (зависит от сложности) |
| Скорость инференс | < 0.1 сек | ~60 сек (на CPU) |
| Обучение | Требуется размеченный датасет | Не требуется |
| Объяснимость | Ключевые слова (TF‑IDF) | Естественно-языковое объяснение |
| Адаптация к новым жанрам | Нужно переобучение | Работает без изменений |

Вывод: классический ML выигрывает в скорости и точности на фиксированном наборе жанров, а LLM полезна для объяснений и работы с новыми категориями.

## Технический стек

| Компонент | Инструмент |
|-----------|------------|
| API | FastAPI + Uvicorn |
| Классический ML | scikit-learn (TF‑IDF), LightGBM |
| LLM | Ollama (Llama 3 8B) |
| Векторизация | TfidfVectorizer (5000 признаков, n-gram 1–2) |
| Данные | [Google Books Dataset (15k книг, 100+ категорий)](https://www.kaggle.com/datasets/mihikaajayjadhav/books-dataset-15k-books-across-100-categories) |
| Контейнеризация | Docker + docker-compose |
| Деплой | Dockhost (GitHub Actions автосборка) |
| Язык | Python 3.12 |

## Результаты

- Метрика accuracy на тестовой выборке (775 книг):

  77.94% (LightGBM)

- Среднее время ответа:
  
  - `/predict/classic` – ~0.05 сек
  
  - `/predict/llm` – ~60 сек

- Поддерживаемые жанры (топ-10):

  `Fiction`, `Computers`, `Business`, `History`, `Cooking`, `Medical`, `Biography`, `Literary Criticism`, `Psychology`, `Education`
